Dragon showcase — scene_dragon_showcase.txt
==========================================

Three dragons on pedestals (mesh/dragon_simple.obj, ~1.6k faces each, vertex-quantized from dragon_8k).

| Position | Material | Effect |
|----------|----------|--------|
| Left  | Refractive IOR 1.52, delta 0.12 | Dispersion (CLI `dispersion`) |
| Center | Gold GGX Glossy roughness 0.16 | Specular metal |
| Right | Perfect mirror Reflective | Chrome reflection |

Render:
  cd code
  ./build/PA1-2 testcases/scene_dragon_showcase.txt output/final/dragon_showcase.bmp \
      path_mis 128 gamma cuda dispersion
  python3 -c "from PIL import Image; Image.open('output/final/dragon_showcase.bmp').save('results/dragon_showcase.png')"

Note: original dragon.obj has ~871k faces (negative OBJ indices); dragon_simple.obj (~1.6k faces)
keeps total CUDA triangles under 50k for GPU VRAM. dragon_8k.obj (~8.7k faces) also ships as a
coarser alternative.
